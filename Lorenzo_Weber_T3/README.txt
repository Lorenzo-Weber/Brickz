| Como Jogar:	  										   |
| 												   |
| Após compilar o programa e rodar, você entrará no menu do jogo, onde tem a opção de apertar P    |
| para jogar ou Q para sair do jogo, após apertar P temos a tela de controles, as quais definem:   |
|												   |
|	- A para andar para a esquerda e D para direita						   |
|	- Mouse1 para atirar (é um tiro contínuo)						   |
|	- Mouse2 (pressionar) para o jogo rodar 40% mais rapido					   |
|												   |
| e também nos introduzem aos power-ups, os quais são:						   |
|												   |
|	- Max Ammo: Recarrega o canhão, ou seja, se você não estiver atirando, ele não tem efeito; |
|	- Double Points: Duplica os pontos por alvo quebrado;					   |
|	- Juggernog: Sobe os alvos 1 ou 2 vezes	;						   |
|	- Double Tap: Da o dobro de dano;							   |
|	- Speed Cola: Aumenta a velocidade de movimento do canhão;				   |
|	- Quick Revive: Da uma vida extra;							   |
|												   |
| Cada power-up tem uma porcentagem unica de cair, e a chance de cair cada power-up depende	   |
| do nível de dificuldade escolhida pelo jogador (Fácil, Médio ou Difícil), e além disso, a 	   |
| maioria dos power-ups (com excessão do Quick Revive) tem duração dentro da fase em que é	   |
| jogado, ou seja, após uma vitória ou derrota, perde-se todos os power-ups.			   |
| O jogo consiste em rodadas, ou seja, após destruir todos os alvos, você pode ir para uma nova	   |
| fase, e caso perca, poderá ir para ou menu ou restartar tudo.					   |
